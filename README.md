# Сервис для рассылок по почте

## Описание: 
Приложение для отправки рассылок пользователям

## Установка:
1. необходимо скопировать HTTPS ссылку на гит хаб.
2. открыть через PyCharm
```
https://github.com/ArbiKhadjimuradov/coursework4.git
```
3. Установить зависимости 
```
pyproject.toml
```

## Использование:
В терминале python manage.py runserver
Остальное все понятно на сайте
